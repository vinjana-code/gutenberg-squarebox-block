== Description ==

This WordPress Gutenberg block plugin allows users to create a customizable block with the following features:

Heading: Add a heading to your block to provide a title or main text.

Link: Include a hyperlink to direct users to another page or resource.

Background Color Palette: Choose from a selection of background colors to style your block.

Thumbnail Icon: Display a thumbnail icon within the block.

Image Upload/Replace: Easily add or replace an image using the WordPress media library

Features
Customizable Heading:

Add a heading to your block.

Customize the font size, color, and alignment.

Hyperlink:

Add a clickable link to your block.

Customize the link text and URL.

Background Color Palette:

Choose from a variety of background colors.

Use the color palette to match your site's design.

Thumbnail Icon:

Display a thumbnail icon within the block.

Customize the icon size and position.

Image Upload/Replace:

Use the MediaUpload component to add or replace an image.

Display the selected image within the block.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/gutenberg-squarebox-block` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
